How to write an exam

  1. Check the sylabus on what topics the exams cover;This should be done no later than 1 month before the exam.
  2. Study for the exam; cover all the topics on the exam. If there is any part giving you trouble write them down.
  3. Seek help if you are having troble on some topics; whether that is from the lecturer, TA or friends.
  4. Finally, write the exam. Make sure you arrive on time and have all the equipment needed for the exam.
